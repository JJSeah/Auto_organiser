<h3><b>Prerequisites</b></h3>
- A computer with python already installed and configured (Python 3.7 recommended).

  ```shell
  pip install watchdog
  ```
<h3><b>Run</b></h3>
- To run the  program execute the python file by: 

  ```shell

  python time_organiser.py
  ```
